🔍 02. Exploratory Data Analysis (EDA)

Dataset Examples

Original Format: All images in the dataset are in grayscale format and sized exactly 28x28 pixels. They represent 10 distinct types of clothing and accessories.

Class Distribution
Perfectly Balanced: The dataset exhibits an equal distribution across all 10 classes, with exactly 6,000 samples per class in the training set.

Categories include: T-shirt, trouser, pullover, dress, coat, sandal, sneaker, bag, shirt, and ankle boot.

⚙️ Setup & Preprocessing Pipeline

  • Resize: Scale images from 28x28 to 224x224 to increase the input resolution required by modern CNN architectures.
  • Channel Conversion: Convert 1-channel grayscale images to 3-channel RGB format to match standard pre-trained model input shapes.
  • Normalization: Normalize pixel values to a standard range to improve model training stability and convergence speed.

🏗️ 03. Model Building Pipeline

💡 Approach: We compare two state-of-the-art architectures for computer vision: Convolutional Neural Networks (CNN) vs. Vision Transformers (ViT).

CNN: DenseNet

Extracts features hierarchically using dense connectivity patterns, where each layer receives inputs from all preceding layers.

Vision Transformer (ViT)

Splits the image into fixed-size patches, linearly embeds them with position encodings, and feeds them into a standard Transformer encoder.

🔄 Data Pipeline & Tensor Shapes

1. Raw Input: (Batch, 1, 28, 28) // Grayscale image batch
2. Preprocessed: (Batch, 3, 224, 224) // Resized & 3-channel expanded
3. Backbone Output: (Batch, Features) // E.g., DenseNet121 yields 1024 dims

🧠 Classification Head

The original classification head of the backbone is replaced with a new Fully Connected (Linear) Layer that maps the high-dimensional feature vector (Batch, Features) down to the final class logits (Batch, 10). The entire network is optimized using Cross-Entropy Loss.

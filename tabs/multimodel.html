02. Exploratory Data Analysis (EDA)

Data Insights

Long-tail Image Distribution: Images per class vary significantly. Cat (3421), Dog (2336), Cow (1353), Sheep (1161), Horse (933).

Setup & Preprocessing

  • Text Prompt Generation: Convert dataset classes into specific structures (e.g., "a photo of a {object}").
  • Prototype Extraction (Few-shot): Extract vector prototypes for each class.

03. Model Building Pipeline

Contrastive Language-Image Pre-training (CLIP)

Zero-shot: Predicts the class by estimating which generated text prompt pairs best.

Few-shot: Evaluates the similarity of a new image's vector against the 5 class vector prototypes.

01. Dataset, EDA & Preprocessing

VNTC (Vietnamese News)

84,132
News Articles
⚖️ 🏛️ 🔬 ⚽

Collected from multiple major online newspapers (VnExpress, Tuổi Trẻ, Thanh Niên, Người Lao Động).

Training (33,759)40%
Testing (50,373)60%

Exploratory Data Analysis

Imbalanced Distribution: Society (Chính trị Xã hội) and Sports (Thể thao) dominate both train and test sets, while Science (Khoa học) is the minority.
  • Categories: 10 Classes including Politics, Lifestyle, Science, Business, Law, Health, World, Sports, Culture, IT.
  • Diverse Styles: Articles range from short blurbs to complex reports.

Data Preprocessing Pipeline

  • Raw Text Handling: Clean and prepare real-world Vietnamese text.
  • Word Segmentation: Utilize tools like VnCoreNLP or underthesea to correctly group Vietnamese compound words (e.g., "học sinh").
  • Subword Tokenization: Apply Byte Pair Encoding (BPE) to tokenize text into subwords, optimizing the input specifically for the PhoBERT architecture.

02. Model Building Pipeline

🎯 Comparison: RNN vs. Transformer

RNN: LSTM (Long Short-Term Memory)

Designed to capture sequential dependencies in text by maintaining a memory cell that can retain information over time (Forget, Input, and Output gates).

Transformer: PhoBERT

Pretrained language model specifically designed for Vietnamese, based on the RoBERTa architecture. Uses a self-attention mechanism to process the entire input sequence simultaneously, capturing long-range dependencies effectively.

01. Dataset, EDA & Preprocessing

MSCOCO Subset

5
Target Classes
🐈 🐄 🐕 🐎 🐑

Provides authentic Image-Text pairs describing the same entity, ensuring multimodal integrity and high complexity.

1 Image 🔗 5 Human-written Captions

Exploratory Data Analysis

Long-tail Image Distribution: Images per class vary significantly. Cat (3421), Dog (2336), Cow (1353), Sheep (1161), Horse (933).
  • Caption Length: Follows a normal distribution, peaking at around 50 characters per caption.
  • Top 20 Words: 'sheep', 'field', 'cows', 'cat', 'grass', 'standing', 'grazing', 'dog', 'white', etc.

Data Setup & Preprocessing Pipeline

  • Text Prompt Generation: Convert dataset classes into specific caption structures (e.g., "a photo of a {object}") for zero-shot capability.
  • Prototype Extraction (Few-shot): Extract vector prototypes for each class (5 vectors for 5 classes) from the image embedding space to use as similarity anchors.

02. Model Building Pipeline

🎯 Approach: CLIP (Zero-shot vs. Few-shot)

Contrastive Language-Image Pre-training (CLIP)

Zero-shot Classification: Predicts the class by estimating which generated text prompt (e.g., "a photo of a dog") pairs best with a given image using contrastive scoring.

Few-shot Classification: Evaluates the similarity of a new image's vector against the 5 pre-extracted class vector prototypes. The class with the highest similarity is selected.
